درباره پروژه

این پروژه با هدف تحلیل و نمایش کلمات و هجای آن‌ها در اشعار فارسی طراحی شده است. کاربران می‌توانند با استفاده از این ابزار به راحتی کلمات مورد نظر خود را جستجو کرده و اطلاعات مربوط به هجای آن‌ها را مشاهده کنند.

ویژگی‌های اصلی پروژه:

  • جستجوی سریع و آسان کلمات و هجای آن‌ها
  • امکان فیلتر کردن کلمات بر اساس وزن و هجای آن‌ها
  • نمایش تعداد کل کلمات و وضعیت تایید آن‌ها
  • رابط کاربری ساده و کاربرپسند

این پروژه با استفاده از HTML، CSS و JavaScript توسعه داده شده است و از فریم‌ورک Tailwind CSS برای طراحی رابط کاربری استفاده می‌کند. همچنین، از فونت‌های فارسی برای نمایش بهتر متن‌ها بهره گرفته شده است.

امیدواریم این ابزار به شما در تحلیل اشعار و درک بهتر زبان فارسی کمک کند.
